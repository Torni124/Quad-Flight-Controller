# Flight controller
attempt at a quadcopter flight controller, stopped when I obliterated my control board at the end of the summer. Achieved extremely unstable vertical flight. No control over yaw due to poor IMU readings from a cheap board. 
Quadcopter consisted of a handmade wooden frame(Terrible but no 3D printer :( ) with 4 2400KV brushless motors 
Control board was an Arduino Nano with an MPU6050 IMU, Communication with a handmade controller(using another Nano) was done via an RF transceiver board.
